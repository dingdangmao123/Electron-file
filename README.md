### 总结
- 功能比较简单，利用web方式构建页面较为方便快速
- node.js 写起来有较大不适，动态类型和各种回调不方便调试，维护不方便，缺乏一个强大的IDE提供智能提示
- 总之用来编写简单的桌面小工具还是比较快捷方便的，工程量不大
